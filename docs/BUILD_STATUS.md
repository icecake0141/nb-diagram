# Build Status

## Current state

- Runtime template first tries `static/dist/app-main.js` and falls back to
  `static/app-main.js` if not found.
- In this environment, Node toolchain (`node`/`npm`) is unavailable.
- Therefore `static/dist/*` is currently populated by direct module copy from `static/*.js`.

## When Node is available

Run:

```bash
npm install
npm run frontend:build
```

Then `static/dist/app-main.js` will be generated by Vite from `frontend/src/*.ts`.

## 日本語訳

### 現在の状態

- ランタイムのテンプレートは、まず `static/dist/app-main.js` を読み込み、
  見つからない場合に `static/app-main.js` へフォールバックします。
- この環境では Node ツールチェーン（`node` / `npm`）が利用できません。
- そのため現在の `static/dist/*` は `static/*.js` を直接コピーして構成しています。

### Node が利用可能な場合

以下を実行します:

```bash
npm install
npm run frontend:build
```

すると `frontend/src/*.ts` から Vite により `static/dist/app-main.js` が生成されます。
